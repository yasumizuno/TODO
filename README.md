# ToDo抽出アプリケーション

このアプリケーションは、アップロードされた文書からルールベースでToDoを抽出し、リスト化するウェブアプリケーションです。

## 機能

- 文書のアップロード（対応形式: TXT, PDF, DOCX, DOC）
- ルールベースでのToDo項目の抽出
- 抽出されたToDo項目のリスト表示
- ToDo項目のクリップボードへのコピー
- ToDo項目のCSVファイルとしてのダウンロード

## ToDo抽出ルール

以下のパターンに一致するテキストをToDo項目として抽出します：

1. "TODO", "TO-DO", "To Do", "ToDo" などの後に続くテキスト
2. Markdown形式のタスク（"- [ ]" や "* [ ]" で始まる行）
3. "タスク:" または "タスク：" の後に続くテキスト
4. "要対応" または "対応必要" を含む行
5. 期限に関する表現（"期限", "締切", "〆切", "デッドライン", "due date", "deadline"）を含む行
6. 日付パターン（YYYY/MM/DD や MM/DD）の後にタスク説明が続く行
7. 日付の後に "まで" が続き、その後にタスク説明がある行（例: "7月31日までに報告書を提出"）
8. 一般的なアクション動詞（"作成する", "準備する", "確認する" など）を含む行
9. "必要" や "必須" を含む行（必要なアクションを示す）
10. "予定" や "スケジュール" の後にタスク説明が続く行

## インストール方法

```bash
# リポジトリをクローン
git clone https://github.com/yasuhiro-mizuno/TODO.git
cd TODO

# 依存パッケージのインストール
pip install -r requirements.txt

# アプリケーションの起動
python app.py
```

## 使用方法

1. ブラウザで `http://localhost:12000` にアクセス
2. 「ファイルを選択」ボタンをクリックして文書ファイルを選択
3. 「アップロード＆抽出」ボタンをクリックして処理を開始
4. 抽出されたToDo項目が表示されます
5. 「すべてコピー」ボタンでクリップボードにコピー、または「CSVダウンロード」ボタンでCSVファイルとしてダウンロード可能

## 対応ファイル形式

- テキストファイル (.txt)
- PDFファイル (.pdf)
- Word文書 (.docx)

## 注意事項

- アップロードされたファイルは処理後に自動的に削除されます
- 大きなファイルの処理には時間がかかる場合があります
